【ゲームタイトル】

   COS/MOS

【コンセプト】

   ひらめきと団結力で楽しめる

【制作期間 / 開発環境】

   10月16日〜1月26日(約4ヶ月間) / VisualStudio 2015 DrectX9

【制作者】

   飯田洲暉・大殿慶輔・大宮愛羅・加賀谷悠貴・菊池陽人
   小室大夢・西村吏功・平澤詩苑・藤原龍輝

【ゲーム内容】

   2人プレイの協力アクションパズルゲーム
   世界観は宇宙をモチーフとしている
   プレイヤーの位置を任意のタイミングで反転可能
   両方の世界に影響を与えるギミックがある

【操作説明】

   [プレイヤー用操作]
   移    動：左スティック/十字ボタン左右
   ジャンプ：Bボタン
   スワップ：Yボタン
   
   [基本操作]
   選    択：左スティック/十字ボタン左右
   決    定：Aボタン
   戻    る：Bボタン

【アピールポイント】

   プレイヤーモデルの腕や触手にスキンメッシュを使用
